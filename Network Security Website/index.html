I ett samhälle där barn i unga åldrar utsätts för internet utan ordentlig utbildning eller erfarenhet är det upp till föräldrarna att säga ifrån och informera. Men vad ska man säga till sitt barn? Vad behöver man skydda sig från och hur skyddar man sig?

Introduktion

Hej! Mitt namn är Daniel Westberg och jag studerar för närvarande datateknik vid Linköpings Universitet. Det har slagit mig att allt fler ungdomar springer omkring oskyddade i en allt mer internet integrerat samhälle. Ett misstag kan leda till stora problem och människor som vill dig illa kan utnyttja internet för att vilseleda och förtrycka. Denna sida vänder sig framförallt till unga och föräldrar som vill vara informerade om hur man kan skydda sig själv och de omkring sig från onödiga risker på internet. Vi kommer gå igenom sånt som nedladdning, safe trade, mobiltelefoner och routrar. Detta projekt ger också mig möjligheten att fördjupa mig inom allmän datasäkerhet. Informationen på denna sida kommer hålla en lägre avancerad nivå. Informationen är uppdelad i artiklar som ni kan se nedan.
